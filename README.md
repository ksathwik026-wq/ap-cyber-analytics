# Andhra Pradesh Cyber Crime Analytics Dashboard
### APSCHE B.Tech CSE Community Service Project (CSP)

A lightweight, responsive web dashboard built to analyze and visualize regional cyber crime and digital fraud trends in Andhra Pradesh using official secondary datasets.

Live URL: https://[your-github-username].github.io/ap-cyber-analytics/

## 📊 Project Overview
* **Domain:** Cybersecurity & Data Visualization
* **Target Region:** Andhra Pradesh, India
* **Data Sources:** National Crime Records Bureau (NCRB) & Indian Cyber Crime Coordination Centre (I4C)
* **Academic Alignment:** Formally fulfills the 4-credit mandatory Community Service Project (CSP) criteria for B.Tech Computer Science & Engineering under the APSCHE guidelines.

## 🛠️ Tech Stack & Architecture
* **Frontend:** Semantic HTML5, Responsive CSS3 Grid/Flexbox
* **Logic Engine:** Vanilla JavaScript (ES6+)
* **Visual Engine:** Chart.js via minified CDN link
* **Environment:** Developed 100% inside a plain-text editor (Notepad) with zero external compilation frameworks.

## ⚡ Core Features
1. **Interactive Analytics:** Canvas-driven line and doughnut charts tracking annual case distribution and primary fraud vectors.
2. **Algorithmic Threat Filter:** A native JS real-time keyword search engine to look up active regional scam patterns (Digital Arrests, UPI fraud).
3. **Deterministic Risk Calculator:** A client-side vulnerability scoring module evaluating user password and authentication habits.

## 📁 Repository Structure
* `index.html` - Application architecture and DOM anchors.
* `style.css` - Custom responsive layout rules and theme sheets.
* `app.js` - Data arrays, Chart.js bindings, and engine event handlers.

## 🚀 Deployment Instructions
1. Fork or clone this repository.
2. Open `index.html` directly in any modern browser to run locally.
3. For cloud hosting, go to Repository Settings -> Pages -> Enable deployment from the `main` branch.
